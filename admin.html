📋 Google Sheets 연동 설정 가이드

STEP 1 — Google Sheets 만들기
1. sheets.google.com 에서 새 스프레드시트 만들기
2. 스프레드시트 이름: Raina's Wine Cellar
3. 시트1 이름을 Purchase 로 변경
4. 1행에 헤더 입력: id | name_kr | type | vintage | store | date | price | memo
5. 시트 추가 → 이름 Tasted 로 설정
6. Tasted 1행: id | name_kr | type | vintage | tasted_date
STEP 2 — Apps Script 만들기
1. 스프레드시트에서 확장 프로그램 → Apps Script 클릭
2. 기존 코드 전체 삭제 후 아래 코드 붙여넣기
3. 저장(💾) 클릭 

            

            
          


          

            
STEP 3 — 웹 앱으로 배포 (⚠️ 권한 설정 중요)

            

              1. Apps Script 편집기 상단 배포 → 새 배포 클릭

              2. ⚙️ 아이콘 클릭 → 웹 앱 선택

              3. 설명: Wine Cellar API

              4. 다음 사용자로 실행: 나 (내 Google 계정)

              5. 액세스 권한: 모든 사용자 (Anyone) ← 반드시 이것!

              6. 배포 → Google 계정 권한 허용 → 웹 앱 URL 복사


              ⚠️ 403 오류가 나는 경우:

              배포 → 배포 관리 → 연필 아이콘 → 액세스를 모든 사용자로 변경 → 새 버전으로 재배포
            

          


          

            
STEP 4 — URL 입력

            

              복사한 URL을 이 페이지 상단의 Google Apps Script URL 칸에 붙여넣기 → 저장
